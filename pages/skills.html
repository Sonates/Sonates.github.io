Java

Java ist eine objektorientierte Programmiersprache und eine der beliebtesten Sprachen der Welt. Die Sprache wurde 1995 von Sun Microsystems entwickelt und ist seit 2010 eine Marke der Oracle Corporation. Java ist eine der einfachsten Sprachen zu erlernen und bietet eine Vielzahl an Funktionen für die Entwicklung von Anwendungen. Ich habe in der Schule grundlegende Kenntnisse der Sprache erworben, aber ich bin weiterhin interessiert an der Sprache und ihren Möglichkeiten.

SQL

Ich habe in der Schule grundlegende Kenntnisse in der Programmiersprache SQL erworben und mein Interesse an der Sprache wurde geweckt. SQL ist eine Abkürzung für Structured Query Language und ist eine standardisierte Sprache, die zur Manipulation und Abfrage von Daten in relationalen Datenbanken verwendet wird. SQL ist eine sehr mächtige Sprache, die mir ermöglicht, meine Kenntnisse der Datenbanken zu vertiefen und mich weiterzuentwickeln.

PHP ist eine Skriptsprache, die ursprünglich für die Erstellung von dynamischen Webseiten konzipiert wurde. PHP-Code kann in HTML-Code eingebettet werden oder als eigenständige Dateien gespeichert werden, die vom Webserver ausgeführt werden. Ich habe in der Schule Grundkenntnisse in PHP erworben, ich werde mich jedoch nochmals mit dem Thema auseinandersetzen, um es komplett zu verstehen.
